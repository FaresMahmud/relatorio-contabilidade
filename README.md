# 📊 Relatório Financeiro — 2024 & 2025

Dashboard financeiro interativo desenvolvido como projeto acadêmico para a disciplina de Contabilidade Gerencial.

🔗 **[Acesse o site](https://faresmahmud.github.io/relatorio-contabilidade/)**

---

## Sobre o Projeto

Aplicação web estática que apresenta as demonstrações financeiras de uma empresa de tecnologia nos exercícios de 2024 e 2025, com visualizações interativas, análise automática dos indicadores e cálculo de índices financeiros.

Os dados foram originalmente implementados em Java (classes `Main.java` e `Relatorios.java`) e convertidos para uma interface visual moderna hospedada via GitHub Pages.

## Funcionalidades

- **Balanço Patrimonial** — Ativo, Passivo e Situação Líquida com detalhamento completo
- **Resultado Operacional** — Receita Bruta, Receita Líquida, Margem de Contribuição e EBITDA
- **Comparativo 2024 vs 2025** — Tabela e gráficos side-by-side com variações percentuais
- **Análise Automática** — Interpretação dos indicadores em linguagem clara
- **Índices Financeiros** — Liquidez Geral, Endividamento, ROA, Margem EBITDA e outros
- **Linha do Tempo** — Visualização dos dois exercícios com marcos principais
- **Barra de Saúde Financeira** — Indicadores visuais de performance
- **Tooltips interativos** — Fórmula e cálculo detalhado em cada card
- **Animações** — Contadores, sparklines e transições suaves
- **Exportar PDF** — Impressão direta pelo navegador

## Tecnologias

- HTML5, CSS3, JavaScript (vanilla)
- [Chart.js 4.4](https://www.chartjs.org/) — gráficos
- [Google Fonts](https://fonts.google.com/) — DM Serif Display + DM Sans
- GitHub Pages — hospedagem
